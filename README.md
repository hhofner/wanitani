# WaniTani

A terminal-based [WaniKani](https://www.wanikani.com) client.

![demo](demo.gif?v=2)

## Install

```
brew install hhofner/tap/wanitani
```

or

```
go install github.com/hhofner/wanitani@latest
```

## Usage

```
wanitani
```

Set your API token with `/add-token`, then use `/learn` or `/review`.
